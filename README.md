# Лендинги для бренда VOLGA (Авторитэйл и Вагнер-Авто)

Премиальные адаптивные посадочные страницы (лендинги) для презентации и продажи нового модельного ряда автомобилей **VOLGA** (седан бизнес-класса **C50**, среднеразмерный кроссовер **K40** и флагманский кроссовер **K50**).

Разработаны две версии посадочных страниц в соответствии с ТЗ:
1. **Москва (Авторитэйл)** — `index.html`
2. **Санкт-Петербург (Вагнер-Авто)** — `spb.html` (с интерактивным переключателем по 3 дилерским центрам)

## 🌟 Ключевые особенности и функционал

- **Премиальный дизайн**: Темная, роскошная цветовая гамма с золотыми (шампань) акцентами, подчеркивающая статус бренда VOLGA.
- **Интерактивный конфигуратор цвета**: Пользователи могут в реальном времени переключать цвета кузова для каждой модели (C50, K40, K50) и видеть автомобиль в выбранном цвете.
- **Интерактивный выбор дилерского центра (для Санкт-Петербурга)**: Динамическое переключение адресов, телефонов и автоматическое обновление интерактивной карты Yandex Maps для выбранного автосалона (Оптиков, Таллинский, Выборгский).
- **Адаптивная верстка**: Идеальное отображение на любых устройствах — от мобильных телефонов до широкоформатных мониторов.
- **Формы захвата лидов**: Современные формы для записи на тест-драйв и получения коммерческого предложения с валидацией, анимацией загрузки и красивым модальным окном успешной отправки.
- **Интеграция с Calltouch**: В код встроен готовый шаблон-интеграция для отслеживания лидов через систему Calltouch.
- **Официальная графика**: Использованы оригинальные высококачественные изображения автомобилей и SVG-логотип бренда VOLGA.

## 🚀 Как запустить проект

1. Склонируйте репозиторий или скачайте файлы.
2. Откройте файл `index.html` (для Москвы) или `spb.html` (для Санкт-Петербурга) в любом современном веб-браузере.
3. Для работы интерактивных карт Yandex Maps требуется активное интернет-подключение.
